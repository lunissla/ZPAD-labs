Лабораторна робота №2



Аналіз даних за допомогою Pandas



Дана лабораторна робота присвячена аналізу даних із використанням бібліотеки Pandas у середовищі Python. Робота складається з двох частин.



У першій частині здійснюється аналіз набору даних VHI (Vegetation Health Index). Виконується завантаження даних, їх попередня обробка, формування вибірок за областями, фільтрація за роками та тижнями, а також обчислення основних статистичних показників.



У другій частині проводиться аналіз набору даних Individual Household Electric Power Consumption Dataset. Реалізовано очищення даних (data cleaning), формування вибірок за заданими умовами, вибір випадкових 500000 записів без повторів та обчислення середніх значень для груп споживання електроенергії. Також виконано нормалізацію та стандартизацію даних, обчислено коефіцієнти кореляції Пірсона та Спірмена для числових атрибутів, проведено One Hot Encoding категоріального атрибуту та здійснено профілювання часу виконання процедур за допомогою модуля timeit.



Для виконання роботи використовуються Python 3, Pandas, NumPy та Jupyter Notebook.



Структура теки лабораторної роботи містить файли 1part.ipynb, 2part.ipynb, requirements.txt та README.md.



Інструкція із запуску:



Клонувати репозиторій за допомогою команди

git clone <URL\_репозиторію>

та перейти до теки лабораторної роботи.



Створити та активувати віртуальне середовище (рекомендовано):

python -m venv venv

venv\\Scripts\\activate



Встановити необхідні залежності:

pip install -r requirements.txt



Запустити Jupyter Notebook командою:

jupyter notebook



Після запуску необхідно відкрити файли 1part.ipynb та 2part.ipynb і виконати всі комірки послідовно.
