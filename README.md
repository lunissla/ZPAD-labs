# Лабораторні роботи з"ЗПАД"

---

## Інформація про мене

**Прізвище та ім’я:** Кисельова Єлизавета
**Група:** ФБ-42 
**Дисципліна:** ЗПАД

---
**Частина 1** — Аналіз даних споживання електроенергії домогосподарств (Individual household electric power consumption dataset)

**Частина 2** — Завантаження, очищення та аналіз Vegetation Health Index (VHI) по областях України (NOAA дані)

## Вимоги

- Python 3.9 або новіший
- ~6–8 GB вільної оперативної пам'яті (для частини 1)
- Інтернет (тільки для першого запуску частини 2)
