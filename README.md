# Лабораторні роботи з"ЗПАД"

---

## Інформація про мене

**Прізвище та ім’я:** Кисельова Єлизавета
**Група:** ФБ-42 
**Дисципліна:** ЗПАД

---

## Опис репозиторію

Цей репозиторій містить лабораторні роботи з курсу "ЗПАД".

Кожна лабораторна робота розміщена в окремій директорії та оформлена відповідно до вимог курсу.
