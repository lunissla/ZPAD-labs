\# Лабораторна робота №3



\## Візуалізація та аналіз даних за допомогою Python



Дана лабораторна робота присвячена аналізу та візуалізації даних із використанням бібліотек Python у середовищі Jupyter Notebook.



У роботі використано набір даних Adult Income Dataset, отриманий з відкритого репозиторію UCI Machine Learning Repository. Датасет є багатовимірним (multivariate) та містить як числові, так і категоріальні атрибути, а також пропущені значення, що дозволяє виконати повноцінний процес підготовки даних.



У першій частині роботи здійснюється завантаження даних та їх попередній аналіз. Виконується очищення даних (data cleaning), зокрема обробка пропущених значень та приведення типів даних до відповідного формату. Також проводиться аналіз структури датасету та визначення основних характеристик.



У другій частині роботи здійснюється візуалізація даних. Побудовано різні типи графіків, зокрема:

\- графіки залежності між числовими змінними (scatter plot, line plot);

\- гістограма розподілу значень з використанням заданих інтервалів;

\- boxplot для аналізу розподілу даних;

\- heatmap для відображення кореляції між числовими змінними;

\- countplot для аналізу категоріальних даних;

\- pairplot для комплексного аналізу взаємозв’язків між змінними.



На основі побудованих графіків виконано аналіз залежностей між змінними, виявлено закономірності у даних та сформульовано відповідні висновки.
